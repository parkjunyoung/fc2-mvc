# Nodejs MVC

### Routing

GET /posts => 글리스트<br />

GET /posts/write => 글작성<br />
POST /posts/write<br />

GET /posts/detail/:id => 상세글 보기<br />

GET /posts/edit/:id => 글수정<br />
POST /posts/edit/:id<br />

GET /posts/delete/:id => 글삭제<br />

GET /posts/ajax_comment/insert => 댓글 추가<br />
GET /posts/ajax_comment/delete => 댓글 삭제<br />


### Model

Post
- title : 글제목
- content : 글 내용
- created_at : 작성일

Comment
- content : 댓글 내용
- created_at : 작성일
- post_id : Post의 id
